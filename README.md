* [unstableanimals.com](https://unstableanimals.com)
* [Twitter](https://twitter.com/UnstableAnimals)
* [Discord](https://discord.gg/dCX6vqxXNm)

## Development

Requires Node.js 18 or newer (see `.nvmrc`).

```bash
yarn install
yarn compile          # compile Solidity contracts into src/artifacts
yarn start            # run the frontend dev server
```

## Build

The frontend imports contract ABIs from `src/artifacts`, which are generated by Hardhat.
`yarn build` runs `hardhat compile` automatically via the `prebuild` script.

```bash
yarn build
```

## Tests

```bash
yarn test:contracts   # Hardhat / Solidity tests
CI=true yarn test --watchAll=false   # React tests
```
